🌱 AI Smart Farming System

📌 Project Overview

The AI Smart Farming System is a web-based application designed to help farmers using simple AI-based logic and automation concepts.
It combines multiple smart farming features into a single platform to improve productivity and efficiency.

---

🎯 Features

🔐 Login System

- User login interface (frontend-based)
- Custom username & password supported

💧 Smart Irrigation

- Detects soil condition
- Suggests watering:
  - Dry → Water needed 💧
  - Moist → No water needed ✅

🌾 Soil Recommendation

- Suggests crops based on soil:
  - Black soil → Cotton, Soybean
  - Red soil → Groundnut
  - Others → Rice

📷 Disease Detection

- Upload plant image (demo feature)
- Shows detection result

🧑‍🌾 Farmer Market

- Farmers can add:
  - Name
  - Crop
  - Price
- Displays selling details

🎤 AI Voice Assistant

- Supports English & Telugu
- Provides voice guidance

🌦️ Weather Info

- Displays temperature (demo)
- Can be upgraded with API
